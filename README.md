# Taverna do Mestre
Site focado em automatização de fichas para RPG, o objetivo central é cumprir uma tarefa acadêmica e expandir para um site real onde a comunidade poderá alterar

Este site contará com usuarios podendo alterar entre mestre e player, onde o mestre será responsável pela criação e gerenciamento da campanha, enquanto o player fará tudo acontecer.
